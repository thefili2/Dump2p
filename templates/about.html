Dump p2p

How it works

Dump is a hybrid chat application combining the persistence of a server-backed lobby with the absolute privacy of Peer-to-Peer (P2P) rooms.

Two Modes of Communication

  • Main Chat (Server): A global lobby where messages are temporarily stored on our server so you can see recent chat history when you join.
  • Private Rooms (P2P): Encrypted, direct browser-to-browser connections. Once the room is closed, the data ceases to exist.

Features

  • Text Chat: Instant messaging in both Main and Private rooms.
  • Voice Channels: Join an opt-in voice channel in Private Rooms. Audio flows directly between users via WebRTC.
  • File Sharing: Share images and documents (up to 5MB) directly in Private Rooms without uploading to any server.

The Technology

  • Backend: Flask & SQLite handle the Main Chat and user presence.
  • P2P Engine: PeerJS & WebRTC handle the Private Rooms, Voice, and Files.
  • Encrypted: All P2P data (text, voice, files) is End-to-End encrypted.
